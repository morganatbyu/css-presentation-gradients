CSS Topic: Gradients

Gradient is a special type of css tag that is a transition between two or more colors.

Types of Gradients

There are different types of gradients that you can use: linear, radial, conic. The most popular are linear and radial. They can be used where an image would go like a background. They work better than a raster image because it is generated by the browser. It can be resized well. The < gradient> tag is supported on most browsers. Internet Explorer has the least amount of support for it.

Linear Gradient

Linear gradient is created by using the function, linear-gradient(). You can choose to have your gradient go from left to right, top to bottom, and even on a diagonal. All you need to do is to establish the angle. Top: “0deg” Bottom: “180deg” Right:“90deg” Diagonally: “45deg”
